# TestDünya

Türkiye odaklı sınav test platformu başlangıç projesi.

## İçerik

- Next.js 15 + React 18
- Modern, koyu temalı bir ana sayfa tasarımı
- YKS, LGS, KPSS, Ehliyet, AÖL, DGS, ALES, YÖKDİL, TUS, DUS odaklı kartlar
- YDS bilerek **dahil edilmedi** (EnglishMeter tarafına ayrıldı)

## Çalıştırma

```bash
npm install
npm run dev
# veya
pnpm install
pnpm dev
```

Sonra tarayıcıdan: http://localhost:3000

Bu sadece ana sayfa şablonudur. Her sınav için `/yks`, `/lgs` vb. sayfaları ve test mantığını daha sonra ekleyebilirsin.
